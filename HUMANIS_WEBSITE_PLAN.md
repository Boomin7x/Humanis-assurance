## Phase 2 — Design System & Theme

> **Design language: "Refined Precision"**
> Clean, authoritative, and trustworthy — the visual identity of an institution that manages serious risk.
> The system is built on three pillars:
> 1. **Discipline** — No decorative gradients, no superfluous shadows. Every visual decision earns its place.
> 2. **Depth through color** — Micro-shadows are colored (brand blue tint at low opacity), never grey/black. They communicate structure, not decoration.
> 3. **Motion with restraint** — Entrances are smooth and purposeful. Nothing bounces. Nothing lingers.
>
> **Border radius:** 6px standard · 8px on cards/inputs · 12px on modals/drawers · 100px on pills/badges
> **Shadows:** Permitted — but only brand-blue tinted, ultra-subtle (see elevation scale below)
> **Gradients:** Permitted only in hero sections and dark section backgrounds — not on buttons or cards

## Design Principles — Quick Reference

> These rules must be respected throughout the entire project. Every new component and page must conform to these.

**Design language: "Refined Precision"** — authoritative, clean, trustworthy. Every visual decision earns its place.

### Typography

| Token | Value | Rule |
|---|---|---|
| Heading font | **Plus Jakarta Sans** | All H1–H6. Weight 700–800 for H1–H3, 600 for H4–H6 |
| Body font | **DM Sans** | All body copy, UI labels, captions, buttons |
| H1 scale | **clamp(2.25rem → 3.75rem)** | Fluid — no media queries needed |
| H2 scale | **clamp(1.75rem → 2.75rem)** | Fluid |
| Body | **1rem / 1.70 line-height** | Generous leading for readability |
| Overline | **0.6875rem / 0.12em tracking / ALL CAPS** | Teal color, section label above headings |

### Color

| Token | Value | Rule |
|---|---|---|
| Primary blue | **#0D5EAF** | All CTAs, navbar scrolled, accents |
| Primary hover | **#0A4A91** | Hover state for all primary elements |
| Teal | **#1A9E75** | Secondary CTAs, success, overline labels, feature highlights |
| Dark navy | **#1C2B45** | Hero background, all headings on light bg, footer |
| Page background | **#F7F9FC** | Default page bg |
| Alt section bg | **#EEF3FA** | Every other section — visual rhythm |
| Card surface | **#FFFFFF** | All card backgrounds |
| Border | **#DDE3EC** | All card borders, dividers, input borders |
| Text primary | **#1C2B45** | All headings |
| Text secondary | **#4A5568** | Body copy, descriptions |
| Text muted | **#718096** | Captions, placeholders, meta text |

### Shape & Depth

| Token | Value | Rule |
|---|---|---|
| Radius (buttons) | **6px** | All buttons |
| Radius (cards/inputs) | **8px** | Cards, text fields, accordions |
| Radius (modals/drawers) | **12px** | All overlays |
| Radius (pills/badges) | **100px** | Chips, language toggle, category badges |
| Shadows | **Blue-tinted only** | `rgba(13, 94, 175, 0.06–0.14)` — never grey/black |
| Card shadow (resting) | **elevation.xs** | Barely visible — structure, not decoration |
| Card shadow (hover) | **elevation.sm** | Appears on hover with translateY(-2px) |
| Gradients | **Hero & dark sections only** | Never on buttons, cards, or UI components |

### Motion

| Token | Value | Rule |
|---|---|---|
| Easing | **cubic-bezier(0.16, 1, 0.3, 1)** | Expo ease-out — all reveals and entrances |
| Hover transitions | **200ms** | Color, border, shadow changes |
| Reveal entrances | **450–500ms** | Scroll-triggered fadeInUp, stagger |
| Stagger delay | **0.08s per card** | Card grids |
| Page transitions | **350ms fade** | Route changes via AnimatePresence |
| Rule | **once: true** | Animations trigger once, never on scroll-back |
| Reduced motion | **Respected** | `@media (prefers-reduced-motion)` collapses all durations to 0.01ms |
